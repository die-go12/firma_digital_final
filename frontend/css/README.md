# Hojas de Estilo (CSS)

Este directorio contiene los archivos de estilo que definen la apariencia visual de la interfaz web.

## Archivos

* **`styles.css`**: Hoja de estilos principal.
    * Define la paleta de colores del proyecto.
    * Controla la maquetación (Layout) del Dashboard y los formularios.
    * Asegura la responsividad básica para visualización en diferentes pantallas.

## Tecnologías
* **CSS3 Vanilla** (Sin preprocesadores ni frameworks pesados).
